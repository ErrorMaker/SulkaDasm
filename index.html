Sulkadasm

Tool for patching Habbo Client

SulkaDasm what is it?

SulkaDasm is a tool to disable security protections on the Habbo Client.

By disabling protection, you can use the client with any server.

Usages

Default usage

By default, the client is patched so that it works with any compatible server with the RC4 protocol.

Command (or juste drag and drop the 'Habbo.swf' on sulkadasm.exe) 
sulkadasm Habbo.swf

Note: 'Habbo.swf' is the name of the file to be patched, replace it by the name of your file present in the same folder of SulkaDasm.

Custom RSA Keys

Use this command to set custom RSA Keys

Command 
sulkadasm Habbo.swf --rsaN=Your_Custom_RSA_N_KEY --rsaE=Your_Custom_RSA_E_KEY
Disable RC4 Protocol

Use this command to completely disable the RC4 protocol.

Command 
sulkadasm Habbo.swf --disableRc4

Note: By disabling the RC4 protocol, you will not need to use RSA keys.
